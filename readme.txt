DJ ROTATOR FOR WORDPRESS

######################################
Instructions and Usage

Step 1. Install the plugin using the plugin manager in WordPress
Step 2. Visit Settings --> DJ Rotator
Step 3. Place the DJ Rotator in your theme by using the template tag <?php drwp(); ?> or the shortcode [djwp]

You may customize the CSS using your theme's style.css or custom.css. You can also take advantage of several
hooks built into this plugin to extended functionality.

djwp_before_header
djwp_after_header
djwp_before_image
djwp_after_image
djwp_before_description
djwp_after_description
